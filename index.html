📋 삭제되는 데이터
  • 계정 정보 (이름, 이메일)
  • 제출한 대화 분석 내역

⏱ 데이터 보관 기간
  • 삭제 요청 후 7일 이내 완전 삭제
  • 법적 의무 보관 데이터 제외 (최대 1년)
🗑 앱에서 직접 삭제하는 방법
  1. 미리톡 앱 실행
  2. 하단 메뉴 → 설정(⚙️) 탭 이동
  3. "계정 관리" 선택
  4. "계정 삭제" 버튼 탭
  5. 확인 후 즉시 삭제 처리
